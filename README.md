# Okanese Learning Center Website

Modern Next.js website for Okanese Learning Center.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy

This project is ready for GitHub and Vercel. Push the folder to a GitHub repository, import it in Vercel, and point the GoDaddy domain DNS to Vercel.

## Notes

The design uses FHQTC-adjacent colors, warm school-focused content, responsive layouts, and Framer Motion scroll animations.
